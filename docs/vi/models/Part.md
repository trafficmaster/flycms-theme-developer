# Part Model
- Part là 1 phần của Subject (vd tập phim, tập truyện...)
- Part chứa nhiều Content và Media

## Attributes

| Attribute             | Type    | Mô tả              |
|-----------------------|---------|--------------------|
| sequence              | integer | Số thứ tự của part |
| title                 | string  | Part title         |
| description           | string  | Part description   |
